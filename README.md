# Markdown编辑器

一个基于Vue2的Markdown编辑器和渲染器，支持LaTeX公式。

## 功能特点

- 左侧编辑，右侧实时预览
- 支持标准Markdown语法
- 支持LaTeX数学公式（行内和块级）
- 支持复制粘贴操作
- 美观的渲染结果展示

## 使用方法

1. 克隆项目到本地
2. 安装依赖：
   ```
   npm install
   ```
3. 运行开发服务器：
   ```
   npm run dev
   ```
4. 构建项目：
   ```
   npm run build
   ```

## 技术栈

- Vue.js 2.x
- Marked (Markdown解析)
- KaTeX (LaTeX渲染)
- Webpack
